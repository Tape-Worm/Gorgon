Requirements:
=========================================================================================================
Please note that you will need the following in order to compile Gorgon, its tools and the examples:
1. Visual Studio 2012 (http://www.microsoft.com/visualstudio/eng/downloads#d-express-windows-desktop)
2. DirectX SDK June 2010 (http://www.microsoft.com/en-us/download/details.aspx?id=6812)
   	-or-
   DirectX (http://www.microsoft.com/en-us/download/details.aspx?id=35) if the debug libraries for Gorgon
   are not required.

.NET 4.5 and DirectX is required to run all samples, tools and anything produced with Gorgon.
If you wish to use the debug version of the Gorgon libraries, install the June 2010 DirectX SDK.

For more assistance go to:
http://www.tape-worm.net
