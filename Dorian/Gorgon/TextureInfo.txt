Textures cannot use D3D11_RESOURCE_MISC_DRAWINDIRECT_ARGS.

1D textures cannot use:
	D3D11_RESOURCE_MISC_TEXTURECUBE
	D3D11_RESOURCE_MISC_BUFFER_ALLOW_RAW_VIEWS
	D3D11_RESOURCE_MISC_BUFFER_STRUCTURED	
	D3D11_RESOURCE_MISC_SHARED	

2D textures cannot use:
	D3D11_RESOURCE_MISC_BUFFER_ALLOW_RAW_VIEWS
	D3D11_RESOURCE_MISC_BUFFER_STRUCTURED
	
3D textures cannot use:
	D3D11_RESOURCE_MISC_TEXTURECUBE
	D3D11_RESOURCE_MISC_BUFFER_ALLOW_RAW_VIEWS
	D3D11_RESOURCE_MISC_BUFFER_STRUCTURED
	D3D11_RESOURCE_MISC_SHARED	


All textures can be used as depth/stencil.
All textures can be used as render target.
