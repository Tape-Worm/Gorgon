Redistribution

So you've written this bang up game and now you need to send it to someone and you're asking "How do I do this?" Well, an installer program like NSIS will help greatly. But what you'll really need is to know what files to include with your game or application. And without further ado:

Prerequisite packages for distribution:

These packages are installers that deliver required files for use by Gorgon and should be included with and executed by your installer package (at the very least your victims need to be informed that they need to install these items before running your application).

  • Visual C++ 2008 SP1 runtime - x86 and/or x64 This is required by the version of SlimDX that Gorgon uses
  • The latest and greatest version of the DirectX runtime - This should be fairly obvious.
  • .NET 3.5 SP1
  • Windows XP or better.

Along with those packages you will need be sure that your applications bundles the following with its own installer:

Assembly (DLL/Executable) files required by Gorgon applications:
  • Gorgon.dll
  • SlimDX.dll

Required if you're using the compressed file system (BZip 2 compressed file system):
  • GorgonBZip2FileSystem.dll
  • ICSharpCode.SharpZipLib.dll

Required if you're using the raw input interface plug-in:
  • GorgonInput.dll

Also, keep in mind that if you're developing for an x64 (64-bit) operating system like Windows XP 64 or Vista x64 that your code will not run on x86 (32-bit) operating systems. This is unlike 32 bit binaries which do run on a x64 operating systems.
