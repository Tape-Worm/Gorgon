Thanks for downloading and installing Gorgon.

Any updates to Gorgon, or support can be had at http://www.tape-worm.net.

Gorgon is a 2D graphics library which features:
  • 3D acceleration in a 2D environment.
  • Sprites, naturally.
  • Text and font manipulation.
  • Pixel/vertex shader support.
  • Virtual file systems.
  • An extensible plug-in system.
  • And several other features...
Gorgon uses SharpDX to interface DirectX to .NET. If you have any questions, suggestions or contributions for Gorgon please email me at The author or you can get help from the wiki or the forums on the website.
