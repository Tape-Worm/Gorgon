Remarks

This method will load an assembly that contains GorgonPlugin types. If the assembly does not contain any types that inherit from GorgonPlugin, then an exception will be raised.

Users should call IsPluginAssembly(String) prior to this method in order to determine whether the assembly can be loaded or not.
