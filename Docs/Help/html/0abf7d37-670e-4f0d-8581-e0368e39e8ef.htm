Thanks for downloading and installing Gorgon.

Gorgon is meant to assist any developer in writing an application. Primarily Gorgon is meant for game development, but can be used for so much more. It is a set of libraries that provides functionality for an assortment of needs. This includes:

  • A graphics library patterened around Direct 3D 11. This uses the SharpDX API to provide access to Direct 3D from .NET

  • A 2D renderer that utilizes the aforementioned graphics library to provide ultra fast manipulation of 2D graphics.

  • A virtual file system based on the PhysFS file system.

  • An input system that allows the a unified interface to input devices (keyboard, mouse, game devices, etc...)

  • A plugin system that allows extensibility of an application or even Gorgon's own functionality (e.g. the input system, virtual file system).

  • An animation system that animates the properties of any type of object.

  • An editor that assists in the creation of various types of content for games and/or graphical functionality.

  • A core library of helpful utility functionality.

Getting started

What's required?

  • .NET 4.5

  • Microsoft Windows Vista SP2 at minimum. Windows 7 - 8.1 are supported and future versions may work as well.

  • Microsoft DirectX 11.

To compile the Gorgon solution file

Microsoft Visual Studio 2012 or better

To use the Gorgon libraries

Any .NET 4.5 enabled language (e.g. Visual Basic .NET) can be used to write an application with Gorgon.
