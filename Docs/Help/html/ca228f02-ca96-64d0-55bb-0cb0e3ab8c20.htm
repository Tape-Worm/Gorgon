Remarks
A flat form theme is a means to apply multiple color settings to a FlatForm window. Themes can be loaded from an xml file or set manually via code. By assigning a theme to a flat form, the colors for that window are automatically set by the colors in the window.

The theme object inherits from the ToolStrip renderer class and will be applied as a visual style for tool bars and menus as well.
