The total size of the data to write, in bytes.
Remarks

This function works like ZeroMemory from the Win32 API. It will write a series of zeroes into the memory pointed at by destination.

Ensure that the size is correct, otherwise a buffer overrun may occur and memory corruption will soon follow.
