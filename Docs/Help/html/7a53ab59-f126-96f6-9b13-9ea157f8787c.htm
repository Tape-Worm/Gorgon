Remarks

This method is used to mount the contents of a physical file system object (such as a folder, or a zip file if the appropriate provider is installed) into a virtual folder in the file system. All folders and files in the physical file system object will be made available under the virtual folder specified by the mountPath parameter.

The mountPath parameter is optional, and if omitted, the contents of the physical file system object will be mounted into the root of the virtual file system. If the mountPath parameter is null (Nothing in VB.Net) or empty, then the mount point will be at the root.
