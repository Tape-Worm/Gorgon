Remarks

This method is used to serialize an array of values of the type T as binary data into a GorgonDataStream object.

Unlike the WriteRange methods, this method will accept types that use marshalled fields (i.e. fields that are decorated with the MarshalAsAttribute). This allows for a greater flexibility when copying data into a binary data stream. If the type does not use marshalling, then the method will still copy the data as-is.

Like the WriteRange methods, this method requires that the type T be a value or primitive type, and be decorated with the StructLayoutAttribute with a Value of Sequential (preferred), or Explicit.

 Caution

This method has poor performance. It is not recommended for use except in cases where marshalling must be used to transfer data (e.g. a String or Array needs copying to a C/C++ buffer).

For best performance, use only primitive types or value types (that, in turn, use only primitive/value types) within the value type and use the WriteRange methods.

Examples
An example where an array of values will be copied into a new stream object:
C#
Copy
[StructLayout(LayoutKind.Sequential)]
public struct MyStruct
{
    public int MyInt;
    [MarshalAs(UnmanagedType.ByValTStr, SizeConst = 256)]
    public string MyStr;
}

MyStruct[] items = new MyStruct[50];

for (int i = 0; i < items.Length; ++i)
{
    items[i] = new MyStruct
            {
                MyInt = i,
                MyStr = "String #" + i
            };
}

using (var newStream = GorgonDataStream.MarshalArrayToStream(items))
{
    // Do your work with you stream data.
}
