Remarks

This method will retrieve a list of fully qualified type names for plugins contained within the GorgonPluginAssemblyCache passed to this object. This list is not indicative of whether the type has been created or not.

The assemblyName parameter, when not null (Nothing in VB.Net), will return only plugin names belonging to that assembly. If the assembly is not loaded, then an exception is thrown.
