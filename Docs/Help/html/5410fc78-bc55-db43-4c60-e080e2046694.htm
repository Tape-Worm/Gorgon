Remarks

If the file pointed at by assemblyFile contains GorgonPlugin types, then this method will retrieve a list of plugin names from that assembly. This method can and should be used to determine if the plugin assembly actually contains any plugins, or to retrieve a catalog of available plugins.

The assembly being enumerated is not loaded into the current application domain, and as such, is not cached.

If the file pointed by the assemblyFile parameter is not a .NET assembly, then an exception will be raised.

Since this method takes a path to an assembly, the SearchPaths property will be used if the assembly could not be found on the path specified.

Users should call IsPluginAssembly(String) prior to this method in order to determine whether the assembly can be loaded or not.
