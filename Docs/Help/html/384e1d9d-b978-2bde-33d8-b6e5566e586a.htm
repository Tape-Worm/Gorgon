Usage Note

In Visual Basic and C#, you can call this method as an instance method on any object of type ListView. When you use instance method syntax to call this method, omit the first parameter. For more information, see Extension Methods (Visual Basic) or Extension Methods (C# Programming Guide).
Exceptions
ExceptionCondition
GorgonExceptionThrown if the column header was not found or could not be updated.
Remarks
Use this extension method to set a sorting icon for the specified column. This will give users a clue as to how the list view is sorted.
