Remarks

By default, this object is initialized with multiple search paths, in order of preference. These search paths are:

  1. The directory of the executable.
  2. The working directory of the executable.
  3. The system directory.
  4. The directories listed in the PATH environment variable.

Users are free to add and remove paths from this list as they see fit, but that calling the Clear method will remove these paths. In order to refresh these paths, call GetDefaultPaths.

This collection is -not- thread safe.
