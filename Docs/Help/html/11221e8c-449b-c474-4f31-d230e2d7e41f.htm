Remarks
Remarks

The stream passed to this method requires that the CanSeek property returns a value of true.

The appSpecificIds parameter is an IEnumerableT because there may be multiple versions of the file that an application might wish to read. By providing a list of which versions are supported through the application specific IDs, the reader can determine if the file type is readable or not.
