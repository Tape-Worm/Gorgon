DirectoryNotFoundExceptionThrown when the directory in the path parameter was not found.
Remarks
This method will accept file name masks like file*, file??1 and file*a* when searching.

Please note that the fileMask is not a path. It is the name (or mask of the name) of the file we wish to find. Specifying something like /MyDir/ThisDir/C*w.ext will fail.
