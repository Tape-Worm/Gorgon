Remarks
Use this to replace the Application.Run(new Form()) method in the startup function.

The application uses an idle loop method to call the users code when it is running. A form may also be assigned as the primary form for the application.

An application is started by calling its Run method. An application can be shut down by calling its Quit method. Applications with a main form will end when the form is closed. Alternatively, the application can be terminated by returning false from the idle loop method.

Any objects created in Gorgon, such as the Graphics interface, will be destroyed when the application ends.
