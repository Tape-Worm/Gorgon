Gorgon.Native

This contains functionality for access to native (unmanaged) memory.

While .NET has a fairly comprehensive set of tools to access native memory, and C# has a pointer type, these are limited and in some cases suffer from extremely poor performance (e.g. the pointer type in C# cannot be used with generic types). Therefore, these types are provided for incredibly fast access to native memory with as little restriction as possible.

Disclaimer time

The types in here are inherently dangerous to use because they offer little in the way of protection from poor memory management habits. Buffer overruns, memory leaks, etc.. are all possible with these types. USE AT YOUR OWN RISK. The garbage collector will not save you. If you have no idea how to use native memory, pointers, etc... then it's recommended that you do not use these types.
