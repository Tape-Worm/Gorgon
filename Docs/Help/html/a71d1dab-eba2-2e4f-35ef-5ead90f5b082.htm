Remarks

This property will intercept an event on the current application domain to resolve assembly dependencies as they are loaded. This is necessary to handle issues where types won't load or instantiate in an assembly at run time.

For example, if a custom type converter attribute is specified in a plugin assembly, it may not instantiate unless some assemblies are resolved at load time. Setting this property with a method that will look up assemblies in the current application domain will correct the issue.
