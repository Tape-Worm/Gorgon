Remarks

This method will load the assembly into a separate AppDomain and will determine if it contains any types that inherit from GorgonPlugin. If the assembly does not contain any plugin types, then this method returns false, otherwise it will return true.

Users should call this method before calling Load(AssemblyName) to determine whether or not a plugin assembly should be loaded into the application.

Because this method loads the assembly into a separate application domain, the assembly will not be cached.
