The IGorgonPluginService type exposes the following members.

Methods
  NameDescription
GetPluginT
Function to retrieve a plugin by its fully qualified type name.
GetPluginNames
Function to retrieve a list of names for available plugins.
GetPluginsT
Function to retrieve the list of plugins from a given assembly.
ScanPlugins
Function to scan for plugins in the loaded plugin assemblies that are cached in the GorgonPluginAssemblyCache passed to this object.
Unload
Function to unload a plugin by its name.
UnloadAll
Function to unload all the plugins.
