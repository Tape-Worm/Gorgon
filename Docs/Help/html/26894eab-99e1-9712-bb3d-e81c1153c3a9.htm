Remarks

Please refer to one of the GorgonPointer types in the see also section for more information.

 Caution

Do not use this class to build custom pointer types. This class may change at any time in the future and cause major problems. Instead, use the IGorgonPointer interface if implementing a custom pointer type.

If this class is used for inheritance, then no support will be given if something breaks.
