GorgonConcurrentDictionaryTK, TV Class

[This is preliminary documentation and is subject to change.]

A custom concurrent dictionary that supplements the original ConcurrentDictionaryTKey, TValue type by supplying a read-only interface.

Remarks
This type is the same as the ConcurrentDictionaryTKey, TValue type, with the only difference being that it supports the IReadOnlyDictionaryTKey, TValue interface. See the documentation on ConcurrentDictionaryTKey, TValue for more information.
