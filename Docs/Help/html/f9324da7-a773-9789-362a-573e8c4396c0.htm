Remarks

This will set the values for each property marked with the GorgonApplicationSettingAttribute to the default value for the property type. If the attribute has defined the DefaultValue property, then that value is used instead.

For dictionaries and lists, their contents will be cleared if the property value is not null (Nothing in VB.Net). For array types, the array values will be reset to the default value for the element type.
