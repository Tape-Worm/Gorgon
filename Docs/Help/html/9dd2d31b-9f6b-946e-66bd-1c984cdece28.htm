SlimMath Namespace

[This is preliminary documentation and is subject to change.]

SlimMath is a mathematics API used extensively by Gorgon. It contains many useful functions to perform various calculations for linear algebra and beyond.

SlimMath was created by Mike Popoloski and is located at https://code.google.com/p/slimmath/

(Please note, at the time of this writing, Google is shutting down Google Code. Therefore, this URL may not be valid upon release).

Classes

Gjk
Implements the Gilbert-Johnson-Keerthi algorithm for convex polyhedra collision detection.
